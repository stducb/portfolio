LOG2990 : Projet de logiciel d'application Web

Le second projet intégrateur était axé sur la conception, la réalisation, la validation et le test d'une application Web. L'objectif était de recréer Microsoft Paint sur une page Web en utilisant Angular. Il fallait donc reproduire tous les différents outils disponibles. Cela incluait donc le crayon de base, mais aussi les outils de copier-coller, la commande de retour en arrière et l'agrandissement de la zone de dessin. Dans le cadre du développement, il a aussi fallu tester notre application à l'aide de Jasmine, un framework adapté au développement et à l'exécution de tests unitaires automatisables. Finalement, la possibilité d'enregister les dessins et de reprendre un ancien dessin devrait être implémenté. POur cela, nous avons utilisé une base de données MongoDB.

Dans le cadre de ce projet, j'ai eu l'opportunité de :

  • Développper une application Web

  • Organiser le développement du projet dans une équipe de six personnes

  • Respecter les délais de présentation à chaque mois

Les principales technologies utilisées dans le développement du projet étaient Angular, Typescript, HTML, CSS, Jasmine et MongoDB.
